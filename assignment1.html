Assignment 1

Mission Statement

The aim of my website is to provide a good web novel ecosystem for authors and readers.

Goals

  1. Provide web novel readers with a pleasent reading experience.
  2. Provide web novel authors with a platform to share their work.
  3. Create a website with a good user experience and user interface.

Layout Design

I would like to model a layout similar to webnovel.com. I like the design because it looks clean, minimalistic, and has a nice aesthetic. The website also deals with novels, so I will have a good reference point to how things might be structured and presented.


Content & Pages

Home

The home page will be where users can discover trending novels. There will be a selection of top recommended weekly/monthly novels.


Novel Page

There will be a novel page for each novel. It will contain a description of the novel, some statstics, and a list of chapters.


Chapter Page

There will be a chapter page for each chapter in each novel. The page will be designed to provide a comfortable reading experience.


Author Page

This page will show details about an author. It will have a list of all novels written by this author that are available on the site.


About/Contact/Guidelines

This page will display information about this website. A method to contact the website owner and guideliens for the website.

Target Audience

The target audience is anyone who reads or writes web novels. A significant portion of readers use mobile devices, so the website must have a responsive design.
